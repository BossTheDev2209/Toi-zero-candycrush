# TOIZero

Single-user, Mastercard-themed practice app for Thailand Olympiad in Informatics (TOI) preparation.

See [docs/superpowers/plans/2026-05-25-toizero-app.md](docs/superpowers/plans/2026-05-25-toizero-app.md) for the implementation plan and [info/DESIGN.md](info/DESIGN.md) for the visual design spec.

## Quick start

```
bun install
cp settings.example.json settings.json
bun run dev:server   # one terminal
bun run dev:web      # another terminal
```

Then open http://localhost:5173.
